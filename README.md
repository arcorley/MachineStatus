# Machine Status

A macOS menu bar app that monitors system metrics in real time.

## Features

- CPU usage
- Memory usage
- Disk usage
- GPU stats
- Battery status
- Network activity
- Top processes

## Requirements

- macOS 14+
- Xcode 16+ / Swift 6

## Building

Open `MachineStatus/MachineStatus.xcodeproj` in Xcode and build, or use Swift Package Manager:

```bash
cd MachineStatus
swift build
```
